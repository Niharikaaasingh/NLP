# 🎭 Emotion Prediction using NLP

This project applies **Natural Language Processing (NLP)** techniques to predict emotions from text data.  
It leverages machine learning models to classify sentences into categories such as joy, sadness, anger, fear, and more.

---

## 🔑 Features
- Text preprocessing (tokenization, stopword removal, lemmatization)
- Feature engineering (TF‑IDF, embeddings)
- Model training and evaluation (Logistic Regression, Naive Bayes, Random Forest, etc.)
- Performance metrics: Accuracy, Precision, Recall, F1‑score
- Modular project structure with Google Colab notebooks
- Recruiter‑friendly dataset labeling (`text`, `emotions`)

---
